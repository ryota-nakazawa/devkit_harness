# 使い方（非エンジニア向け）

## これは何？

一言で要望を言うだけで、Claude Code が**動くプロトタイプ**を作ってくれるハーネスです。
技術の知識は一切不要です。

## 使い方

### 1. Claude Code をこのフォルダで起動

```bash
cd prototype-harness
claude
```

### 2. 作りたいものを一言で伝える

例：
- 「カフェのホームページを作って」
- 「お問い合わせフォームが欲しい」
- 「AIと雑談できるチャットを作りたい」

### 3. 5つの質問に答える

Claude が以下のような質問を順番に出します。全部選択式です。

1. 誰が使う？（お客さん／社内／自分）
2. 何をしたい？（見せる／入力してもらう／AI と話す）
3. データを保存する？（する／しない）
4. 見た目は？（シンプル／カラフル／ビジネス風）
5. 必須の項目は？（タイトル・連絡先など）

### 4. 完成

`output/<プロジェクト名>/index.html` が生成されます。
ダブルクリックするだけでブラウザで開けます。

## よくある質問

**Q. 完成したものを編集したい**
A. 「○○の色を青にして」「タイトルを△△に変えて」と Claude に頼めば修正してくれます。

**Q. エラーが出たら？**
A. Claude が自動で直してくれます。何もしなくて大丈夫です。

**Q. 公開したい**
A. 完成後「ネットに公開したい」と伝えれば、Vercel などでの公開手順を案内します。

**Q. 前に作ったやつを流用したい**
A. 「前回のカフェをベースにパン屋さんにして」のように頼めば、過去の作品を元に新しいものを作れます。履歴は `history/index.jsonl` に記録されています。

**Q. Codex でも使える？**
A. 使えます。ルートの `AGENTS.md` が Codex 向けのエントリポイントです。Claude Code では `CLAUDE.md` が読まれます。中身はほぼ同じで、どちらでも同じプロトタイプが作れます。
